## 🚀 Что это?

**ЭТрН-Бот** — Telegram-бот, который позволяет водителям и логистам **оцифровать бумажные ТТН** прямо с телефона. Бот принимает фото товарно-транспортной накладной, распознает данные с помощью AI (Google Gemini Vision) и автоматически создает черновик **электронной транспортной накладной (ЭТрН)** в системе [Saby (СБИС)](https://saby.ru/).

### Принцип работы: Minimal JSON → Черновик в СБИС

Бот делает только критически важную работу — собирает **обязательные данные** из фото/ручного ввода и отправляет минимальный валидный JSON в API СБИС. Логист видит готовый черновик в Saby TMS и при необходимости добивает опциональные поля (пломбы, температуру, состояние тары) в удобном веб-интерфейсе.

## ✨ Возможности

| Функция | Описание |
|---|---|
| 📸 **AI-распознавание (OCR)** | Фото бумажной ТТН → структурированные данные. Поддержка обеих сторон (лицевая + оборот) |
| 🤖 **Gemini 2.5 Flash** | Детальный промпт с описанием формы 1-Т, температура 0.1 для точности |
| ✅ **Валидация данных** | ИНН по контрольной сумме ФНС, госномер (лат→кир), масса (кг→тонны) |
| 📝 **Ручной ввод** | Пошаговый ввод всех полей через Telegram-интерфейс |
| ✏️ **Редактирование** | Инлайн-правки любого поля перед отправкой |
| 📄 **Интеграция СБИС** | Генерация Титула 1 (грузоотправитель) через `СБИС.СгенерироватьВложение` → `СБИС.ЗаписатьДокумент` |
| 💳 **Монетизация** | Тарифные планы + оплата через ЮKassa (Telegram Payments API) |
| 👑 **Админ-панель** | Управление пользователями, тарифами, лимитами, рассылки |
| 🔐 **Авторизация СБИС** | Каждый пользователь подключает свой аккаунт Saby |

## 📦 Стек технологий

- **Runtime**: Node.js 18+
- **Telegram**: [Telegraf](https://telegraf.js.org/) v4
- **AI/OCR**: [Google Generative AI](https://ai.google.dev/) (Gemini 2.5 Flash)
- **БД**: PostgreSQL
- **ЭДО**: СБИС (Saby) JSON-RPC API
- **Платежи**: ЮKassa через Telegram Payments

## ⚡ Быстрый старт

### 1. Клонируй репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/ettn-bot.git
cd ettn-bot
```

### 2. Установи зависимости

```bash
npm install
```

### 3. Настрой окружение

```bash
cp .env.example .env
# Заполни .env своими ключами (см. ниже)
```

### 4. Создай БД PostgreSQL

```sql
CREATE DATABASE ttn_avto;
```

> Таблицы создаются автоматически при первом запуске бота.

### 5. Запусти

```bash
node index.js
```

## 🔑 Переменные окружения

| Переменная | Описание | Где получить |
|---|---|---|
| `BOT_TOKEN` | Токен Telegram-бота | [@BotFather](https://t.me/BotFather) |
| `GEMINI_KEY` | API-ключ Google Gemini | [Google AI Studio](https://aistudio.google.com/) |
| `DB_USER` | Пользователь PostgreSQL | Локальная настройка |
| `DB_HOST` | Хост БД | `localhost` для локальной разработки |
| `DB_NAME` | Имя базы данных | По умолчанию: `ttn_avto` |
| `DB_PASSWORD` | Пароль БД | Локальная настройка |
| `DB_PORT` | Порт PostgreSQL | По умолчанию: `5432` |
| `PAYMENT_TEST_TOKEN` | Токен ЮKassa для Telegram Payments | [ЮKassa](https://yookassa.ru/) |

## 📱 Флоу использования

```
1. /start → Главное меню
2. "Создать ЭТрН" → Выбор типа ТС (тент, реф, контейнер...)
3. Выбор режима: 📸 Фото ТТН | ✍️ Ручной ввод
4. [Фото] → Отправка фото лицевой стороны → (опционально) оборот
5. AI распознает данные → Показ черновика с валидацией
6. Редактирование (если нужно) → Подтверждение
7. Отправка в СБИС → Черновик ЭТрН создан!
```
